# ◈ Quelldex

**源码与数据的秩序引擎** — 轻量级文件整理 + 内置版本控制 + 看板计划 + 数据可视化 桌面工具

> *Quelldex = 德语 Quell「源」+ dex「索引」*

[English](README.md) | 简体中文

---

## 为什么选 Quelldex？

大多数文件管理器不懂你的代码，大多数 IDE 不帮你整理数据。Quelldex 弥补了这个空白——自动将文件分为 9 大类、内置完整版本控制（不需要 Git）、零配置可视化 CSV/JSON 数据、还有按分支隔离的看板任务管理。

**5,300 行 Python，仅依赖 PySide6，跨平台运行。**

---

## 核心功能

### 智能文件浏览
- **9 类自动分类** — 源代码 · 数据 · 文档 · 配置 · 脚本 · 样式 · 图片 · 输出 · 其他
- **3 种视图** — 分类视图 · 目录树 · 平铺列表
- **实时搜索** — 去抖筛选，大项目不卡
- **暂存架 (Smart Shelf)** — 批量操作先预览后执行
- **固定文件 & 最近文件** — 快速访问常用文件

### 内置版本控制
完整的 VCS 引擎，无需外部依赖，适合不用 Git 的项目。

- **快照提交** — SHA-256 哈希寻址对象存储
- **分支管理** — 创建 / 切换 / 合并 / 删除
- **标签快照** — 标记重要版本，一键回溯
- **差异对比** — 逐行 Diff，增删改三色高亮
- **变更追踪** — 实时检测新增、修改、删除
- **合并冲突检测**

### 计划看板 (Planner)
项目任务管理器直接嵌入文件组织工具，无需切换上下文。

- **三列看板** — 待办 → 进行中 → 已完成
- **分支作用域** — 任务可绑定到共享（所有分支）或某个具体分支
- **作用域过滤** — 一键切换查看范围
- **分支合并** — 跨分支复制任务，按标题智能去重
- **四级优先级** — 紧急（红）· 高（金）· 中（蓝）· 低（灰）
- **丰富卡片** — 标题、描述、标签、截止日期、优先级指示

### 数据可视化
零配置图表，支持 CSV / TSV / JSON，QPainter 原生绘制。

- **6 种图表** — 直方图 · 折线图 · 柱状图 · 散点图 · 相关矩阵 · 多文件叠加
- **多文件对比** — 多份数据在同一坐标轴上绘制
- **完整统计** — 均值、中位数、标准差、方差、偏度、Q1/Q3、频率

### 文件对比
- **文件 vs 文件** — 任意两个文件逐行对比
- **文件 vs 版本** — 当前文件与 VCS 历史版本对比

### 多主题系统
- **3 套主题** — 暗色 · 亮色 · 午夜
- **运行时切换** — 无需重启，偏好自动保存

### IDE 与工具集成
- **自动检测** — VS Code、Cursor、Sublime Text、Vim
- **智能双击** — 代码文件自动调 IDE，数据文件自动开可视化
- **外部工具协议** — JSON 配置注册自定义工具

## 快速开始

```bash
pip install PySide6
python main.py
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 打开项目文件夹 |
| `Ctrl+S` | 快速提交版本 |
| `Ctrl+1~6` | 切换六个视图 |
| `Ctrl+Tab` | 在多项目间轮换 |
| `F5` | 强制刷新（清除缓存） |

## 技术栈

- Python 3.9+
- PySide6 / Qt6
- QPainter 原生图表（无 matplotlib）
- SHA-256 内容寻址存储
- 纯 Python PNG 生成器（`struct` + `zlib`）
